# ぐっと睨む

## すべてのジョブを入力・実行・出力で表す

|       | 入力                           | 実行                         | 出力     |
| ----- | ------------------------------ | ---------------------------- | -------- |
| build | `SUBMISSION=fn main() { ... }` | `cargo build` など | バイナリ |
| run   | バイナリ、`STDIN=1 2`          | `./worker` など              | `3`      |

## すべての入出力を Artifacts で表す

- 入力はすべて Artifacts のダウンロードで行う。
- 実行はすべて Artifacts をマウントした Docker 上で行う。
- 出力はすべて Artifacts のアップロードで行う。
